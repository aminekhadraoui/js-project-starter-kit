Tag Reference

<repeat>
UI Component
Repeats HTML content with unique attributes for each iteration. Supports nested elements and attribute propagation.
<repeat iteration="3" name="item_{n}" class="card card-{n}"> <div class="{class}"> <h3>Item {n}</h3> <input name="{name}"> </div> </repeat>
<slider>
UI Component
Creates an image slider/carousel with configurable options and smooth transitions.
<slider images="img1.jpg,img2.jpg" interval="3000" controls="true" height="400px"> </slider>
<cart>
UI Component
Creates responsive product cards with hover effects. Supports multiple cards with text, subtitles, and images.
<cart text="Product 1,Product 2" soustext="Desc 1,Desc 2" image="img1.jpg,img2.jpg" direction="horizontal" price="19.99,29.99"> </cart>
<formGroup>
UI Component
Creates a complete form with configurable attributes. Includes optional submit button with modern styling.
<formGroup method="post" action="/submit.php" id="my-form" class="custom-form" color="#f72585" button="on"> <!-- Inputs go here --> </formGroup>
<inputGroup>
UI Component
Generates multiple form inputs based on comma-separated attribute values. Automatically creates labels and inputs.
<inputGroup name="name,email" class="input-field,input-field" type="text,email" id="name-input,email-input" require="true,true"> </inputGroup>
<columns>
Backend
Defines table columns with data types, constraints, and key definitions. Used within create operations.
<columns> <column name="id" type="int" primaryKey="true" /> <column name="name" type="varchar" length="100" /> </columns>
<operation>
Backend
Defines CRUD operations with SQL queries and column definitions. Enhanced with column specifications.
<operation type="create" table="users"> <columns> <column name="id" type="int" primaryKey="true"/> <column name="email" type="varchar" length="255"/> </columns> INSERT INTO users (email) VALUES (:email) </operation>
